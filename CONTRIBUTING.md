# Panduan Kontribusi

Terima kasih telah berminat berkontribusi pada proyek Internal Audit FAPA!

## Cara Berkontribusi

### 1. Fork Repository
Klik tombol "Fork" di halaman repository GitHub ini.

### 2. Clone Repository Lokal
```bash
git clone https://github.com/username/Internal-Audit-FAPA.git
cd Internal-Audit-FAPA
```

### 3. Buat Branch Baru
```bash
git checkout -b feature/nama-fitur
```

### 4. Buat Perubahan
Lakukan perubahan yang diperlukan dan commit dengan pesan yang jelas.

### 5. Push ke GitHub
```bash
git push origin feature/nama-fitur
```

### 6. Buat Pull Request
Buka Pull Request dari branch Anda ke branch `master` di repository utama.

## Standar Commit

- Gunakan imperative mood ("Add feature" bukan "Added feature")
- Singkat dan deskriptif
- Contoh: `Add documentation for audit process`

## Pertanyaan?

Hubungi tim audit untuk bantuan lebih lanjut.
